# CodeVector

CodeVector.com

## About CodeVector
CodeVector is an E-learning web application for online learning and teaching.

## To Run the server 

#### `npm i`
#### `npm run server`

## Features

- Login and Sign up
- Carousel & Search Results
- Sorting on product Page
- Add to cart
- Apply the coupon on your cart 
- Purchase the Product
- Fullscreen mode
